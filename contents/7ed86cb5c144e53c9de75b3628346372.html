Enviar Ordem de Compra

UMLUseCase
Untitled :: Model :: Enviar Ordem de Compra

Description

Descrição: Este caso de uso descreve o processo pelo qual um usuário pode enviar uma ordem de compra para adquirir ações por meio do sistema Home Broker da Soy Stocks (S2). A funcionalidade permite que os usuários participem ativamente do mercado de ações, indicando o desejo de comprar um determinado número de ações de uma empresa listada na B3.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em comprar ações específicas por meio da plataforma Home Broker. Soy Stocks (S2): Interessada em facilitar as transações de compra de ações e fornecer uma interface eficiente e segura para os usuários.

Pré-condições: O usuário deve estar autenticado no sistema. O usuário deve ter fundos disponíveis em sua conta para cobrir o custo da ordem de compra.

Fluxo Básico: O usuário acessa a seção de compra no Home Broker. O usuário seleciona a ação desejada e especifica a quantidade de ações que deseja comprar. O sistema valida a disponibilidade de fundos na conta do usuário. Se houver fundos suficientes, o sistema gera uma ordem de compra contendo os detalhes fornecidos pelo usuário. O usuário revisa os detalhes da ordem de compra. O usuário confirma e envia a ordem de compra. O sistema processa a ordem de compra e registra a transação no histórico da conta do usuário. O usuário recebe uma confirmação da ordem de compra.

Fluxos Alternativos: Se o usuário não tiver fundos suficientes em sua conta, o sistema notifica sobre a impossibilidade de processar a ordem de compra. O sistema pode permitir que o usuário defina parâmetros adicionais, como preço limite, na ordem de compra.

Pós-condições: A ordem de compra é registrada no sistema e a transação é processada.

Regras de Negócio: O sistema pode oferecer informações em tempo real sobre a cotação atual da ação para auxiliar o usuário na tomada de decisão. As ordens de compra podem ser sujeitas a taxas de corretagem, conforme as políticas da Soy Stocks (S2).

Exceções: Se houver problemas técnicos durante o processo de envio da ordem de compra, o sistema deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Notas Adicionais: Essa funcionalidade é central para a experiência do usuário no Home Broker, permitindo a participação ativa no mercado de ações. Medidas de segurança, como confirmação da ordem, são importantes para evitar transações não intencionais.
