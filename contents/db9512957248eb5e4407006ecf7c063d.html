Criar Grupos de Ações

UMLUseCase
Untitled :: Model :: Criar Grupos de Ações

Description

Descrição: Este caso de uso descreve como o usuário, autenticado no sistema Home Broker, realiza a operação de criar grupos específicos para organizar suas ações dentro da carteira. A funcionalidade de criar grupos de ações oferece ao usuário a capacidade de personalizar a organização de seus ativos de acordo com suas preferências e estratégias de investimento.
