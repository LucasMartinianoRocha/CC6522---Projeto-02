Desconectando

UMLState
Untitled :: StateMachine3 :: (Region) :: Desconectando

Description

O estado "Desconectando" refere-se ao processo pelo qual um usuário encerra ativamente sua sessão ou conexão com o sistema. Este estado é acionado quando um usuário decide sair ou se desconectar voluntariamente do sistema.
