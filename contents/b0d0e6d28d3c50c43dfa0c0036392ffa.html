Cancelar Ordem de Venda

UMLUseCase
Untitled :: Model :: Cancelar Ordem de Venda

Description

Descrição: Este caso de uso descreve o processo pelo qual um usuário, por meio da interface do sistema Home Broker da Soy Stocks (S2), solicita o cancelamento de uma ordem de venda de ações previamente enviada para a Bolsa de Valores de São Paulo (B3). Essa funcionalidade é crucial para permitir que os usuários cancelem ordens de venda antes da execução, caso suas intenções de transação mudem ou ocorram eventos imprevistos no mercado.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em cancelar uma ordem de venda de ações antes de sua execução. Soy Stocks (S2): Interessada em fornecer uma plataforma eficiente e personalizável para que os usuários realizem transações de compra e venda, incluindo a possibilidade de cancelamento de ordens. Bolsa de Valores de São Paulo (B3): Interessada em receber e processar solicitações de cancelamento de ordens de venda de maneira eficiente e transparente.

Pré-condições: O usuário deve estar autenticado no sistema Home Broker. O usuário deve ter uma ordem de venda de ações ativa no sistema Home Broker. O sistema Home Broker deve estar devidamente autenticado e autorizado para solicitar o cancelamento de ordens de venda para a B3.

Fluxo Básico: O usuário acessa a interface do Home Broker e visualiza a lista de suas ordens de venda ativas. O usuário seleciona a ordem de venda que deseja cancelar. O sistema Home Broker verifica se a ordem ainda não foi executada. O usuário confirma a solicitação de cancelamento da ordem de venda. O sistema Home Broker gera a mensagem padronizada para cancelamento e a envia para a Bolsa de Valores (B3) por meio de APIs de integração. A Bolsa de Valores recebe a solicitação, verifica se a ordem ainda é cancelável e processa o cancelamento. A Bolsa de Valores envia uma confirmação do cancelamento de volta ao sistema Home Broker. O sistema Home Broker atualiza o status da ordem de venda no sistema do usuário, refletindo que a ordem foi cancelada com sucesso.

Fluxos Alternativos: Se a verificação da ordem indicar que ela já foi executada, o sistema Home Broker informa ao usuário que a ordem não pode ser cancelada. Se houver problemas técnicos durante o envio da solicitação de cancelamento para a B3, o sistema Home Broker deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Pós-condições: O usuário recebe uma confirmação do sistema Home Broker sobre o cancelamento da ordem de venda. O sistema Home Broker atualiza o histórico de transações do usuário.

Regras de Negócio: O sistema Home Broker deve seguir os padrões e protocolos estabelecidos pela Bolsa de Valores para o cancelamento de ordens de venda. Uma ordem de venda só pode ser cancelada se ainda não foi executada.

Exceções: Se a ordem de venda já foi executada, o sistema Home Broker informa ao usuário que a ordem não pode ser cancelada. Se houver problemas técnicos durante o envio da solicitação de cancelamento para a B3, o sistema Home Broker deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Notas Adicionais: A funcionalidade de cancelamento de ordens proporciona aos usuários maior flexibilidade e controle sobre suas transações, permitindo que ajustem suas estratégias de investimento conforme as condições do mercado e suas necessidades financeiras.
