Consultar Status da OC

UMLUseCase
Untitled :: Model :: Consultar Status da OC

Description

Descrição: Este caso de uso descreve o processo pelo qual um usuário pode consultar o status atual de uma Ordem de Compra (OC) enviada por meio do sistema Home Broker da Soy Stocks (S2). A funcionalidade fornece informações detalhadas sobre o andamento e a execução da OC, permitindo que os usuários acompanhem o progresso de suas transações.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em obter informações atualizadas sobre o status de uma OC enviada. Soy Stocks (S2): Interessada em fornecer uma interface eficiente e segura para os usuários acompanharem o progresso de suas ordens de compra.

Pré-condições: O usuário deve estar autenticado no sistema. Deve existir uma OC previamente enviada pelo usuário.

Fluxo Básico: O usuário acessa a seção de ordens no Home Broker. O usuário visualiza a lista de OCs enviadas e identifica a que deseja consultar. O usuário seleciona a opção "Consultar Status" associada à OC desejada. O sistema exibe informações detalhadas sobre o status da OC, incluindo se ela foi executada, parcialmente executada ou se ainda está pendente. O usuário recebe as informações sobre o status da OC.

Fluxos Alternativos: Se a OC não foi encontrada ou não existe, o sistema informa ao usuário sobre a impossibilidade de consultar o status. Se houver problemas técnicos durante o processo de consulta, o sistema deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Pós-condições: O usuário recebe informações atualizadas sobre o status da OC.

Regras de Negócio: O sistema deve fornecer informações claras e precisas sobre o status da OC. O sistema pode incluir detalhes adicionais, como a quantidade de ações executadas, preço de execução, entre outros.

Exceções: Se houver problemas técnicos durante o processo de consulta, o sistema deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Notas Adicionais: Essa funcionalidade oferece aos usuários a capacidade de acompanhar em tempo real o status de suas OCs, permitindo uma gestão mais eficiente de suas transações no mercado de ações. É essencial fornecer informações claras e atualizadas para garantir a confiança dos usuários no sistema.
