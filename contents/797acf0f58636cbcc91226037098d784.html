Configurar Alertas de Cotação por E-mail ou SMS

UMLUseCase
Untitled :: Model :: Configurar Alertas de Cotação por E-mail ou SMS

Description

Descrição: Este caso de uso descreve como o usuário, autenticado no sistema Home Broker, configura alertas personalizados para cotações de ações. A funcionalidade permite que o usuário receba notificações por e-mail ou SMS quando os preços de ações atingem determinados valores pré-estabelecidos, possibilitando uma resposta rápida a mudanças no mercado.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em monitorar variações nos preços de ações e receber alertas personalizados para tomar decisões oportunas.

Pré-condições: O usuário deve estar autenticado no sistema Home Broker.

Fluxo Básico: O usuário acessa a funcionalidade de configuração de alertas de cotação no sistema Home Broker. O sistema oferece uma interface onde o usuário pode selecionar as ações desejadas, definir os valores de disparo para alertas de alta e baixa, e escolher o método de notificação (e-mail, SMS ou ambos). O usuário configura os parâmetros de alerta de acordo com suas preferências. O sistema armazena as configurações de alerta do usuário.

Fluxos Alternativos: O usuário pode editar ou excluir alertas configurados anteriormente. O sistema pode fornecer opções adicionais, como definir horários específicos para o envio de alertas.

Pós-condições: O usuário "Usuário" está configurado para receber alertas personalizados de cotação de ações com base nos parâmetros definidos.

Regras de Negócio: O sistema deve garantir a entrega segura e oportuna de alertas por e-mail ou SMS, de acordo com as configurações do usuário.

Exceções: Em caso de falha na configuração ou envio de alertas, o sistema deve notificar o usuário e oferecer a opção de corrigir as configurações.

Notas Adicionais: Essa funcionalidade oferece ao usuário a capacidade de monitorar automaticamente as variações de preços de ações que são de seu interesse, permitindo uma gestão mais eficaz de suas estratégias de investimento.
