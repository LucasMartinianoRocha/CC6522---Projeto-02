Cancelar Ordem de Compra

UMLUseCase
Untitled :: Model :: Cancelar Ordem de Compra

Description

Descrição: Este caso de uso descreve o processo pelo qual um usuário pode cancelar uma Ordem de Compra (OC) que foi previamente enviada por meio do sistema Home Broker da Soy Stocks (S2). A funcionalidade permite que os usuários cancelem uma OC antes que ela seja executada, oferecendo flexibilidade no gerenciamento de transações.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em cancelar uma OC que foi enviada, mas ainda não foi executada. Soy Stocks (S2): Interessada em fornecer uma interface eficiente e segura para os usuários gerenciarem suas ordens de compra.

Pré-condições: O usuário deve estar autenticado no sistema. Deve existir uma OC previamente enviada pelo usuário que ainda não foi executada.

Fluxo Básico: O usuário acessa a seção de ordens no Home Broker. O usuário visualiza a lista de OCs enviadas e identifica a que deseja cancelar. O usuário seleciona a opção "Cancelar" associada à OC desejada. O sistema confirma se a OC pode ser cancelada (ou seja, se ela ainda não foi executada). Se a OC pode ser cancelada, o sistema a remove do conjunto de ordens ativas e registra o cancelamento. O usuário recebe uma confirmação do cancelamento da OC.

Fluxos Alternativos: Se a OC já foi executada, o sistema informa ao usuário que não é possível cancelá-la. Se houver problemas técnicos durante o processo de cancelamento, o sistema deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Pós-condições: A OC é removida do conjunto de ordens ativas, indicando que foi cancelada com sucesso.

Regras de Negócio: O sistema deve permitir o cancelamento apenas de OCs que ainda não foram executadas. O sistema pode oferecer informações adicionais sobre o status da OC antes do cancelamento.

Exceções: Se houver problemas técnicos durante o processo de cancelamento, o sistema deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Notas Adicionais: Essa funcionalidade proporciona aos usuários a capacidade de ajustar suas estratégias de investimento cancelando ordens que não desejam mais executar. É crucial garantir que o cancelamento seja possível apenas para OCs que ainda não foram executadas.
