Bloquear Acesso após 3 tentativas sem sucesso

Description

Descrição: Este caso de uso descreve o processo pelo qual o sistema da Soy Stocks (S2) bloqueia o acesso de um usuário à sua conta após três tentativas consecutivas de login sem sucesso. Essa medida visa proteger a conta contra tentativas de acesso não autorizadas ou ataques de força bruta, reforçando a segurança do sistema.

Ator Principal: Soy Stocks (S2)

Stakeholders e Interesses: Usuário: Impactado pela medida de segurança, sendo bloqueado temporariamente após três tentativas falhas. Soy Stocks (S2): Interessada em proteger as contas dos usuários e prevenir atividades não autorizadas.

Pré-condições: O usuário deve ter uma conta no sistema. O usuário deve ter falhado em três tentativas consecutivas de login.

Fluxo Básico: O usuário tenta realizar o login no sistema. O sistema verifica a validade das credenciais fornecidas. Se as credenciais são inválidas, o sistema contabiliza a tentativa de login sem sucesso. O sistema repete os passos 1-3 para cada tentativa subsequente. Após a terceira tentativa consecutiva sem sucesso, o sistema bloqueia temporariamente o acesso à conta associada ao usuário. O usuário recebe uma notificação informando sobre o bloqueio temporário.

Fluxos Alternativos: Se o usuário inserir corretamente suas credenciais em qualquer uma das três tentativas, o contador de tentativas é reiniciado.

Pós-condições: O acesso à conta do usuário é bloqueado temporariamente após três tentativas consecutivas de login sem sucesso.

Regras de Negócio: O período de bloqueio temporário pode ser definido pelas políticas internas da Soy Stocks (S2). O sistema pode fornecer uma opção para desbloquear a conta por meio de procedimentos de recuperação.

Exceções: Em caso de problemas técnicos durante o processo de bloqueio temporário, o sistema deve informar ao usuário sobre a impossibilidade temporária de acessar e sugerir tentar novamente mais tarde.

Notas Adicionais: Essa medida de segurança ajuda a mitigar riscos relacionados a tentativas de login não autorizadas, protegendo as contas dos usuários contra ataques maliciosos.
