Visualizar Extrato das Ações com todos os negócios realizados em um período

UMLUseCase
Untitled :: Model :: Visualizar Extrato das Ações com todos os negócios realizados em um período

Description

Descrição: Este caso de uso descreve como o usuário, autenticado no sistema Home Broker, visualiza um extrato detalhado de todas as transações realizadas para as ações presentes em sua carteira dentro de um período específico. A funcionalidade de visualização de extrato fornece ao usuário um resumo abrangente de todas as operações executadas no mercado financeiro.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em acompanhar e analisar todas as transações realizadas para as ações em sua carteira durante um período determinado.

Pré-condições: O usuário deve estar autenticado no sistema Home Broker.

Fluxo Básico: O usuário acessa a funcionalidade de visualização de extrato no sistema Home Broker. O sistema exibe uma interface onde o usuário pode selecionar o período desejado para o extrato. O usuário especifica o intervalo de datas desejado para visualizar as transações. O sistema recupera e apresenta um extrato detalhado, incluindo todas as transações (compra, venda, cancelamento, modificação) realizadas para as ações na carteira durante o período especificado.

Fluxos Alternativos: O usuário pode aplicar filtros adicionais, como ações específicas ou tipos de transação, para refinar a visualização do extrato.

Pós-condições: O usuário "Usuário" tem acesso a um extrato detalhado de todas as transações realizadas para as ações em sua carteira durante o período selecionado.

Regras de Negócio: O sistema Home Broker deve recuperar com precisão todas as transações relacionadas às ações presentes na carteira do usuário no período especificado.

Exceções: Em caso de falha na obtenção ou processamento dos dados do extrato, o sistema deve exibir uma mensagem informativa ao usuário e oferecer a opção de tentar novamente.

Notas Adicionais: Essa funcionalidade fornece ao usuário "Usuário" uma visão abrangente de todas as atividades de negociação relacionadas às ações em sua carteira, permitindo análises detalhadas e tomada de decisões informadas.
