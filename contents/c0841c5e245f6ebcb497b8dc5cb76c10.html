Enviar OV a Mercado

UMLUseCase
Untitled :: Model :: Enviar OV a Mercado

Description

Descrição: Este caso de uso descreve o processo pelo qual um usuário, por meio da interface do sistema Home Broker da Soy Stocks (S2), envia uma ordem de venda de ações para a Bolsa de Valores de São Paulo (B3) com a especificação de venda "a mercado". Essa funcionalidade é crucial para permitir que os usuários realizem transações de venda de ações imediatamente, aceitando o melhor preço de compra disponível no mercado.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em vender suas ações imediatamente ao melhor preço de mercado por meio do Home Broker. Soy Stocks (S2): Interessada em fornecer uma plataforma eficiente e personalizável para que os usuários realizem transações de venda de ações. Bolsa de Valores de São Paulo (B3): Interessada em receber e processar ordens de venda "a mercado" de maneira eficiente e transparente.

Pré-condições: O usuário deve estar autenticado no sistema Home Broker. O usuário deve possuir as ações que pretende vender. O sistema Home Broker deve estar devidamente autenticado e autorizado para enviar ordens de venda para a B3.

Fluxo Básico: O usuário acessa a interface do Home Broker e inicia o processo de venda de suas ações, selecionando a opção "a mercado" para a ordem de venda. O sistema Home Broker valida a ordem, verificando se todos os parâmetros necessários estão corretos e se o usuário possui as ações que pretende vender. O sistema gera a mensagem padronizada para a ordem de venda "a mercado" e a envia para a Bolsa de Valores (B3) por meio de APIs de integração. A Bolsa de Valores recebe a ordem, processa-a imediatamente e a executa ao melhor preço disponível no mercado. A Bolsa de Valores envia uma confirmação de execução da ordem de volta ao sistema Home Broker. O sistema Home Broker atualiza o status da ordem no sistema do usuário, refletindo que a ordem foi executada com sucesso.

Fluxos Alternativos: Se a validação da ordem falhar, o sistema Home Broker informa ao usuário sobre os problemas encontrados e não envia a ordem de venda para a B3. Se houver problemas técnicos durante o envio da ordem de venda para a B3, o sistema Home Broker deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Pós-condições: O usuário recebe uma confirmação do sistema Home Broker sobre o status da ordem de venda. O sistema Home Broker atualiza o histórico de transações do usuário.

Regras de Negócio: O sistema Home Broker deve seguir os padrões e protocolos estabelecidos pela Bolsa de Valores para o envio de ordens de venda "a mercado". A validação da ordem de venda deve ser rigorosa para garantir que apenas ordens válidas sejam enviadas para a B3.

Exceções: Se a validação da ordem de venda falhar, o sistema Home Broker deve informar ao usuário sobre os problemas encontrados e impedir o envio da ordem. Se houver problemas técnicos durante o envio da ordem de venda para a B3, o sistema Home Broker deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Notas Adicionais: A funcionalidade "a mercado" permite que o usuário venda suas ações imediatamente, aceitando o melhor preço disponível no mercado no momento da transação. Essa opção é adequada para usuários que buscam uma execução rápida de suas ordens de venda.
