Ativando

UMLState
Untitled :: StateMachine3 :: (Region) :: Ativando

Description

O estado "Ativando" refere-se ao processo pelo qual um usuário tem seu acesso ao sistema restaurado ou ativado. Esse estado geralmente é acionado quando um usuário, que estava anteriormente inativo ou bloqueado, solicita ou passa por um processo de ativação para recuperar o acesso ao sistema.
