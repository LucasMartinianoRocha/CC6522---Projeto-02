Exibir Maiores Altas e Maiores Baixas

UMLUseCase
Untitled :: Model :: Exibir Maiores Altas e Maiores Baixas

Description

Descrição: Este caso de uso descreve como o sistema Home Broker apresenta as informações das maiores altas e maiores baixas do mercado financeiro ao usuário autenticado. A funcionalidade permite que o usuário tenha acesso rápido e fácil às variações significativas nos valores das ações, auxiliando na tomada de decisões informadas.

Ator Principal: Soy Stocks (S2)

Stakeholders e Interesses: Usuário: Interessado em acompanhar as maiores altas e maiores baixas do mercado para identificar oportunidades ou gerenciar riscos.

Pré-condições: O usuário deve estar autenticado no sistema Home Broker.

Fluxo Básico: O usuário acessa a função de exibição de maiores altas e maiores baixas no sistema Home Broker. O sistema consulta e apresenta uma lista das ações que apresentaram as maiores variações positivas (altas) e negativas (baixas) em um determinado período de tempo. O usuário visualiza as informações, incluindo o nome da ação, variação percentual, e outros dados relevantes. O sistema atualiza periodicamente as informações para refletir as mudanças no mercado financeiro.

Fluxos Alternativos: O usuário pode aplicar filtros para especificar o período de tempo desejado ou definir critérios específicos para as maiores altas e maiores baixas.

Pós-condições: O usuário "Usuário" tem acesso às informações atualizadas das maiores altas e maiores baixas do mercado.

Regras de Negócio: O sistema Home Broker deve obter dados precisos e atualizados das variações percentuais das ações para apresentar nas maiores altas e maiores baixas.

Exceções: Em caso de falha na obtenção ou processamento dos dados, o sistema deve exibir uma mensagem informativa ao usuário e oferecer a opção de tentar novamente.

Notas Adicionais: Essa funcionalidade oferece ao usuário "Usuário" uma visão rápida das mudanças significativas no mercado, permitindo que ele tome decisões informadas com base nas variações percentuais das ações.
