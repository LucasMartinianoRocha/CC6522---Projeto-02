Visualizar OV relacionadas a uma ação selecionada

UMLUseCase
Untitled :: Model :: Visualizar OV relacionadas a uma ação selecionada

Description

Descrição: Este caso de uso descreve o processo pelo qual um usuário, por meio da interface do sistema Home Broker da Soy Stocks (S2), visualiza as ordens de venda relacionadas a uma ação específica. Essa funcionalidade permite que os usuários obtenham insights sobre as atividades de venda de ações relacionadas a um determinado símbolo, incluindo informações sobre quantidade de ações, preços e status de execução.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em obter informações detalhadas sobre as ordens de venda associadas a uma ação específica para tomar decisões informadas. Soy Stocks (S2): Interessada em fornecer uma interface intuitiva e eficiente para que os usuários acessem informações relevantes sobre as atividades de venda de ações. Bolsa de Valores de São Paulo (B3): Interessada em garantir que as informações sobre as ordens de venda estejam precisas e atualizadas.

Pré-condições: O usuário deve estar autenticado no sistema Home Broker. Existem ordens de venda ativas ou históricas associadas à ação selecionada.

Fluxo Básico: O usuário acessa a interface do Home Broker e seleciona a opção de visualizar ordens de venda relacionadas a uma ação. O sistema Home Broker exibe uma lista das ações disponíveis para seleção. O usuário escolhe a ação específica de interesse. O sistema Home Broker recupera as informações das ordens de venda associadas à ação selecionada por meio de APIs de integração com a Bolsa de Valores (B3). O sistema Home Broker apresenta ao usuário as ordens de venda relacionadas à ação escolhida, incluindo detalhes como datas, quantidades de ações, preços e status de execução.

Fluxos Alternativos: Se não houver ordens de venda associadas à ação selecionada, o sistema Home Broker informa ao usuário que não há informações disponíveis. Se houver problemas técnicos durante a recuperação das informações da Bolsa de Valores (B3), o sistema Home Broker deve informar ao usuário sobre a impossibilidade temporária de exibir as informações.

Pós-condições: O usuário visualiza com sucesso as ordens de venda relacionadas à ação selecionada. O sistema Home Broker mantém a integridade e a precisão das informações apresentadas.

Regras de Negócio: O sistema Home Broker deve seguir os padrões e protocolos estabelecidos pela Bolsa de Valores para recuperar e apresentar as ordens de venda relacionadas a uma ação.

Exceções: Se não houver ordens de venda associadas à ação selecionada, o sistema Home Broker informa ao usuário que não há informações disponíveis. Se houver problemas técnicos durante a recuperação das informações da Bolsa de Valores (B3), o sistema Home Broker deve informar ao usuário sobre a impossibilidade temporária de exibir as informações.

Notas Adicionais: Essa funcionalidade oferece aos usuários uma visão específica das atividades de venda de ações associadas a um símbolo escolhido, auxiliando na análise e na tomada de decisões relacionadas a transações específicas.
