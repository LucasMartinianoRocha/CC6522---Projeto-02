Criar "Guia do Iniciante" para Novos Usuários

UMLUseCase
Untitled :: Model :: Criar "Guia do Iniciante" para Novos Usuários

Description

Descrição: Este caso de uso descreve como o sistema Home Broker, representando a corretora Soy Stocks (S2), oferece um guia informativo para novos usuários que estão ingressando na plataforma. O "Guia do Iniciante" visa proporcionar orientações e informações essenciais para que os novos usuários compreendam as funcionalidades do Home Broker e possam iniciar suas atividades de investimento de maneira eficaz.

Ator Principal: Sistema Soy Stocks (S2)

Stakeholders e Interesses: Usuários Novos: Interessados em compreender as operações e funcionalidades básicas do Home Broker para iniciar suas atividades de investimento.

Pré-condições: O usuário novo acessa a plataforma do Home Broker pela primeira vez.

Fluxo Básico: O sistema identifica que o usuário é novo na plataforma, seja por meio de um processo de registro ou pela primeira autenticação. O sistema apresenta automaticamente o "Guia do Iniciante" na interface do usuário, oferecendo informações sobre: Cadastro de usuário. Processo de login. Navegação na plataforma. Funcionalidades básicas, como consulta de cotações, envio de ordens, entre outros. Recursos de suporte e ajuda disponíveis. O usuário tem a opção de percorrer o guia completo ou acessar seções específicas conforme suas necessidades. Após a leitura do guia, o usuário tem a opção de fechar o guia ou continuar explorando a plataforma.

Fluxos Alternativos: O sistema pode oferecer uma versão interativa do guia, com tutoriais práticos e simulações para reforçar o aprendizado. O guia pode ser acessado a qualquer momento na interface do usuário para referência futura.

Pós-condições: O usuário novo adquire conhecimento básico sobre as operações e funcionalidades do Home Broker por meio do "Guia do Iniciante".

Regras de Negócio: O conteúdo do guia deve ser claro, conciso e adaptado para facilitar a compreensão de usuários iniciantes.

Exceções: Em caso de problemas técnicos que impeçam a apresentação do guia, o sistema deve fornecer uma mensagem informativa e alternativas para obter suporte.

Notas Adicionais: O "Guia do Iniciante" é uma ferramenta essencial para proporcionar uma experiência de usuário amigável e ajudar os novos investidores a se familiarizarem rapidamente com a plataforma do Home Broker.
