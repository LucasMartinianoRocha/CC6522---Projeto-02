Consultar Status da Ordem de Venda

UMLUseCase
Untitled :: Model :: Consultar Status da Ordem de Venda

Description

Descrição: Este caso de uso descreve o processo pelo qual um usuário, por meio da interface do sistema Home Broker da Soy Stocks (S2), verifica o status atual de uma ordem de venda de ações previamente enviada para a Bolsa de Valores de São Paulo (B3). Essa funcionalidade fornece aos usuários informações em tempo real sobre o andamento de suas ordens de venda, como se foram executadas, parcialmente executadas ou ainda estão pendentes.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em obter informações atualizadas sobre o status de suas ordens de venda de ações. Soy Stocks (S2): Interessada em fornecer uma interface transparente e eficiente para que os usuários monitorem o status de suas transações. Bolsa de Valores de São Paulo (B3): Interessada em disponibilizar informações precisas sobre o status das ordens de venda em tempo real.

Pré-condições: O usuário deve estar autenticado no sistema Home Broker. O usuário deve ter uma ordem de venda de ações ativa no sistema Home Broker.

Fluxo Básico: O usuário acessa a interface do Home Broker e seleciona a opção de consultar o status de suas ordens de venda. O sistema Home Broker exibe uma lista das ordens de venda ativas do usuário. O usuário seleciona a ordem de venda específica para a qual deseja verificar o status. O sistema Home Broker consulta a Bolsa de Valores (B3) por meio de APIs de integração para obter o status mais recente da ordem de venda. A Bolsa de Valores retorna as informações atualizadas sobre o status da ordem de venda. O sistema Home Broker exibe as informações ao usuário, indicando se a ordem foi executada, parcialmente executada ou ainda está pendente.

Fluxos Alternativos: Se houver problemas técnicos durante a consulta ao status da ordem na B3, o sistema Home Broker deve informar ao usuário sobre a impossibilidade temporária de exibir as informações mais recentes.

Pós-condições: O usuário recebe informações atualizadas sobre o status de suas ordens de venda. O sistema Home Broker mantém um registro atualizado do status das transações do usuário.

Regras de Negócio: O sistema Home Broker deve seguir os padrões e protocolos estabelecidos pela Bolsa de Valores para consultar o status das ordens de venda.

Exceções: Se houver problemas técnicos durante a consulta ao status da ordem na B3, o sistema Home Broker deve informar ao usuário sobre a impossibilidade temporária de exibir as informações mais recentes.

Notas Adicionais: Esta funcionalidade fornece aos usuários uma visão clara e em tempo real do status de suas ordens de venda, permitindo que tomem decisões informadas com base nas condições do mercado e no andamento de suas transações.
