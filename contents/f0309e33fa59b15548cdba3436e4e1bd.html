Enviar Ordem para a B3

UMLUseCase
Untitled :: Model :: Enviar Ordem para a B3

Description

Descrição: Este caso de uso descreve o processo pelo qual o sistema Home Broker da Soy Stocks (S2) envia uma ordem de compra ou venda para a Bolsa de Valores de São Paulo (B3). Essa funcionalidade é crucial para a execução eficiente e segura das transações no mercado financeiro.

Ator Principal: Soy Stocks (S2)

Stakeholders e Interesses: Sistema Home Broker (S2): Interessado em realizar a integração eficiente e segura com a Bolsa de Valores para enviar ordens de compra ou venda. Bolsa de Valores de São Paulo (B3): Interessada em receber e processar as ordens enviadas pelo sistema Home Broker de maneira confiável.

Pré-condições: O sistema Home Broker deve estar devidamente autenticado e autorizado para enviar ordens para a B3. O usuário deve ter enviado uma ordem de compra ou venda por meio da interface do Home Broker.

Fluxo Básico: O sistema Home Broker recebe uma ordem de compra ou venda do usuário. O sistema valida a ordem, verificando se todos os parâmetros necessários estão corretos e se o usuário tem os fundos ou ações necessários para a transação. O sistema gera a mensagem padronizada para a ordem e a envia para a Bolsa de Valores (B3) por meio de APIs de integração. A Bolsa de Valores recebe a ordem, valida-a e a processa conforme as regras do mercado. A Bolsa de Valores envia uma confirmação de execução ou rejeição da ordem de volta ao sistema Home Broker. O sistema Home Broker atualiza o status da ordem no sistema do usuário, refletindo se a ordem foi executada ou rejeitada.

Fluxos Alternativos: Se a validação da ordem falhar, o sistema Home Broker informa ao usuário sobre os problemas encontrados e não envia a ordem para a B3. Se houver problemas técnicos durante o envio da ordem para a B3, o sistema Home Broker deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Pós-condições: O status da ordem no sistema do usuário é atualizado com base na resposta recebida da Bolsa de Valores (B3).

Regras de Negócio: O sistema Home Broker deve seguir os padrões e protocolos estabelecidos pela Bolsa de Valores para o envio de ordens. A validação da ordem deve ser rigorosa para garantir que apenas ordens válidas sejam enviadas para a B3.

Exceções: Se a validação da ordem falhar, o sistema Home Broker deve informar ao usuário sobre os problemas encontrados e impedir o envio da ordem. Se houver problemas técnicos durante o envio da ordem para a B3, o sistema Home Broker deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Notas Adicionais: Esta funcionalidade é fundamental para a execução eficiente e segura das transações no mercado financeiro. A comunicação eficaz entre o sistema Home Broker e a Bolsa de Valores é crucial para a integridade do processo de negociação.
