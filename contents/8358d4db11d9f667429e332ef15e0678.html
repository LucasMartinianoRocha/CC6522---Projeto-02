Consultar Histórico de OC

UMLUseCase
Untitled :: Model :: Consultar Histórico de OC

Description

Descrição: Este caso de uso descreve o processo pelo qual um usuário pode consultar o histórico completo de suas Ordens de Compra (OC) enviadas por meio do sistema Home Broker da Soy Stocks (S2). A funcionalidade fornece ao usuário uma visão abrangente de todas as transações passadas, permitindo o rastreamento e a análise de suas atividades de investimento.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em revisar e analisar todas as suas OCs anteriores, incluindo detalhes como data, quantidade, preço, e status de execução. Soy Stocks (S2): Interessada em fornecer uma interface eficiente e segura para os usuários acessarem e revisarem seu histórico de transações.

Pré-condições: O usuário deve estar autenticado no sistema.

Fluxo Básico: O usuário acessa a seção de histórico no Home Broker. O usuário seleciona a opção "Consultar Histórico de OC". O sistema exibe uma lista de todas as OCs anteriores do usuário, organizadas por data ou outra métrica relevante. O usuário pode aplicar filtros ou parâmetros de busca para refinar a visualização do histórico, como período de tempo específico, ações específicas, entre outros. O sistema exibe detalhes completos de cada OC no histórico, incluindo data, quantidade de ações, preço, status de execução, entre outros. O usuário pode navegar pelo histórico e revisar as informações conforme necessário.

Fluxos Alternativos: Se o histórico estiver vazio ou não contiver informações relevantes, o sistema informa ao usuário sobre a ausência de dados. Se houver problemas técnicos durante o processo de consulta do histórico, o sistema deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Pós-condições: O usuário tem acesso ao histórico completo de suas OCs, podendo revisar e analisar suas atividades passadas no mercado de ações.

Regras de Negócio: O sistema deve fornecer opções de filtro e ordenação para facilitar a busca e análise do histórico. O sistema pode permitir a exportação do histórico para formatos comuns, como CSV ou PDF, para fins de análise externa.

Exceções: Se houver problemas técnicos durante o processo de consulta do histórico, o sistema deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Notas Adicionais: Essa funcionalidade é essencial para que os usuários possam revisar e aprender com suas experiências passadas no mercado de ações, contribuindo para uma tomada de decisão mais informada no futuro. É crucial garantir uma navegação intuitiva e eficiente no histórico.
