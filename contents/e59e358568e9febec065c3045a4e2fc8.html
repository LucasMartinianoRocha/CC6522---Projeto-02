Personalizar Visualização de Cotações

UMLUseCase
Untitled :: Model :: Personalizar Visualização de Cotações

Description

Descrição: Este caso de uso descreve como o usuário, autenticado no sistema Home Broker , realiza a operação de personalizar a forma como as cotações das ações são apresentadas em sua interface. A funcionalidade oferece flexibilidade ao usuário para ajustar a exibição das informações de acordo com suas preferências e necessidades de análise.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em personalizar a forma como as cotações das ações são apresentadas em sua interface, facilitando a análise e tomada de decisões de investimento.

Pré-condições: O usuário deve estar autenticado no sistema Home Broker com o nome de usuário "Usuário".

Fluxo Básico: O usuário acessa a interface de configurações ou personalização no sistema Home Broker. O sistema Home Broker exibe opções de personalização relacionadas à visualização de cotações. O usuário configura preferências, como escolha de indicadores, layout da informação, cores, entre outros. As personalizações são aplicadas à interface de visualização de cotações.

Fluxos Alternativos: O usuário pode optar por reverter as personalizações para as configurações padrão.

Pós-condições: O usuário realizou com sucesso a personalização da visualização de cotações, ajustando a apresentação das informações de acordo com suas preferências.

Regras de Negócio: O sistema Home Broker deve oferecer opções abrangentes de personalização para atender às diversas preferências dos usuários. As personalizações devem ser aplicadas de forma clara e imediata na interface de visualização de cotações.

Exceções: Se houver uma falha técnica durante o processo de personalização, o usuário pode ser notificado sobre a impossibilidade temporária de aplicar as configurações.

Notas Adicionais: A funcionalidade de personalizar a visualização de cotações proporciona uma experiência mais adaptada às preferências individuais dos usuários, permitindo uma análise mais eficiente e personalizada do mercado de ações.
