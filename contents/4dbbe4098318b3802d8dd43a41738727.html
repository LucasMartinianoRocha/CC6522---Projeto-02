Obter Cotação das Ações na Carteira

UMLUseCase
Untitled :: Model :: Obter Cotação das Ações na Carteira

Description

Descrição: Este caso de uso descreve como o sistema Home Broker, operado pela corretora Soy Stocks (S2), atende à operação de obter as cotações atualizadas das ações presentes na carteira do usuário. A funcionalidade permite que o usuário visualize os valores mais recentes dos ativos, auxiliando na tomada de decisões de investimento.

Ator Principal: Soy Stocks - S2

Stakeholders e Interesses: Usuário: Interessado em obter informações atualizadas sobre o valor das ações presentes em sua carteira de investimentos.

Pré-condições: O usuário deve estar autenticado no sistema Home Broker.

Fluxo Básico: O sistema Home Broker recebe a requisição do usuário para obter as cotações das ações em sua carteira. O sistema verifica as ações presentes na carteira do usuário. Para cada ação, o sistema consulta a Bolsa de Valores de São Paulo (B3) para obter a cotação mais recente. As cotações atualizadas são apresentadas ao usuário, seja em uma interface específica ou integradas à visualização da carteira.

Fluxos Alternativos: Se houver uma falha na obtenção das cotações de algumas ações, o sistema pode notificar o usuário sobre a impossibilidade temporária de acessar determinadas informações.

Pós-condições: O usuário obteve com sucesso as cotações atualizadas das ações presentes em sua carteira.

Regras de Negócio: O sistema Home Broker deve ser capaz de consultar a B3 para obter informações de cotação em tempo real. As cotações devem ser apresentadas de forma clara e organizada para o usuário.

Exceções: Se houver uma falha técnica durante a obtenção das cotações, o usuário pode ser notificado sobre a impossibilidade temporária de acessar as informações.

Notas Adicionais: A funcionalidade de obter cotações das ações na carteira é essencial para que o usuário possa tomar decisões informadas sobre seus investimentos, garantindo uma experiência de investimento transparente e atualizada.
