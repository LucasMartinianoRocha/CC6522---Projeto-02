Visualizar OC relacionadas a uma ação selecionada

UMLUseCase
Untitled :: Model :: Visualizar OC relacionadas a uma ação selecionada

Description

Descrição: Este caso de uso descreve o processo pelo qual um usuário pode visualizar todas as Ordens de Compra (OC) relacionadas a uma ação específica por meio do sistema Home Broker da Soy Stocks (S2). A funcionalidade fornece uma visão detalhada das transações de compra associadas a uma determinada ação, permitindo que o usuário compreenda o histórico de mercado dessa ação.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em obter informações detalhadas sobre todas as OCs associadas a uma ação específica. Soy Stocks (S2): Interessada em fornecer uma interface eficiente e segura para os usuários acessarem informações específicas sobre as OCs relacionadas a uma ação.

Pré-condições: O usuário deve estar autenticado no sistema. Deve haver pelo menos uma OC associada à ação selecionada.

Fluxo Básico: O usuário acessa a seção de informações da ação no Home Broker. O usuário seleciona a opção "Visualizar OCs Relacionadas". O sistema exibe uma lista de todas as OCs de compra associadas à ação selecionada. O usuário pode ver detalhes de cada OC, incluindo data, quantidade de ações, preço, status de execução, entre outros. O usuário pode navegar pela lista e revisar as informações conforme necessário.

Fluxos Alternativos: Se não houver OC associada à ação selecionada, o sistema informa ao usuário sobre a ausência de dados relevantes. Se houver problemas técnicos durante o processo de visualização das OCs relacionadas, o sistema deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Pós-condições: O usuário tem acesso às informações detalhadas de todas as OCs associadas à ação selecionada.

Regras de Negócio: O sistema deve fornecer uma representação clara e organizada das OCs relacionadas, facilitando a compreensão do histórico de mercado da ação. O sistema pode incluir opções de filtragem ou ordenação para aprimorar a visualização das OCs.

Exceções: Se houver problemas técnicos durante o processo de visualização das OCs relacionadas, o sistema deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Notas Adicionais: Essa funcionalidade é valiosa para os usuários que desejam entender o comportamento de mercado de uma ação específica, fornecendo transparência e insights sobre as transações de compra associadas a essa ação. É crucial garantir uma apresentação clara e concisa das informações.
