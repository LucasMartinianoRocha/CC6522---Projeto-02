Aguardando Atualização

UMLState
Untitled :: StateMachine8 :: (Region) :: Aguardando Atualização

Description

Este estado reflete o momento em que a carteira está aguardando a atualização das informações sobre as ações e negociações. Nesse estado, a carteira está temporariamente inativa até que os dados mais recentes estejam disponíveis.
