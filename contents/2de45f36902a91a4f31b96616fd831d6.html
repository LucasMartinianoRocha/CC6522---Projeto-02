Consultando Extrato

UMLState
Untitled :: StateMachine14 :: (Region) :: Consultando Extrato

Description

Este estado representa o momento em que o sistema está respondendo a uma solicitação de consulta ao extrato por parte do usuário. Durante esse estado, o sistema recupera e apresenta as informações do extrato conforme solicitado, oferecendo uma visão resumida ou detalhada das transações realizadas.
