Emitir Aviso caso Cliente esteja negociando no Mercado Fracionário

UMLUseCase
Untitled :: Model :: Emitir Aviso caso Cliente esteja negociando no Mercado Fracionário

Description

Descrição: Este caso de uso descreve como o sistema Home Broker, ao identificar que o usuário está realizando negociações no mercado fracionário, emite um aviso para alertar o cliente sobre essa condição. O mercado fracionário refere-se à negociação de frações de ações, e o aviso busca informar o cliente sobre a natureza específica dessas transações.

Ator Principal: Soy Stocks (S2)

Stakeholders e Interesses: Usuário: Cliente que está realizando negociações no mercado financeiro e pode ser impactado pelo aviso sobre o mercado fracionário.

Pré-condições: O usuário deve estar autenticado no sistema Home Broker. O sistema Home Broker deve monitorar as transações do cliente no mercado financeiro.

Fluxo Básico: O sistema Home Broker detecta que o usuário "Usuário" está realizando negociações no mercado fracionário. O sistema gera automaticamente um aviso informativo sobre a natureza das transações no mercado fracionário. O aviso é exibido de forma clara e visível para o usuário "Usuário" na interface do Home Broker.

Fluxos Alternativos: Se o cliente já estiver ciente das características do mercado fracionário, o aviso pode ser configurado para não ser exibido.

Pós-condições: O usuário "Usuário" é alertado sobre o fato de estar negociando no mercado fracionário, recebendo informações adicionais sobre essa modalidade de transação.

Regras de Negócio: O sistema Home Broker deve monitorar as transações do cliente e identificar quando estas ocorrem no mercado fracionário. O aviso deve ser apresentado de forma clara e compreensível ao usuário.

Exceções: Não há exceções previstas neste caso de uso.

Notas Adicionais: Este caso de uso visa proporcionar ao usuário "Usuário" uma informação relevante sobre o tipo de mercado em que suas transações estão ocorrendo, auxiliando na compreensão do contexto de suas negociações.
