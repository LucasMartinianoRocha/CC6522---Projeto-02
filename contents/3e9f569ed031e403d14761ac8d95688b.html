Alterar Senha

UMLUseCase
Untitled :: Model :: Alterar Senha

Description

Descrição: Este caso de uso descreve o processo pelo qual um usuário pode alterar a senha associada à sua conta no sistema da Soy Stocks (S2). A capacidade de alterar a senha é fundamental para a segurança da conta, permitindo que os usuários atualizem regularmente suas credenciais e garantam a proteção adequada contra acessos não autorizados.

Ator Principal: Usuário

Stakeholders e Interesses: Usuário: Interessado em manter a segurança da sua conta, alterando a senha conforme necessário. Soy Stocks (S2): Interessada em fornecer aos usuários meios seguros para modificar suas senhas.

Pré-condições: O usuário deve estar autenticado no sistema.

Fluxo Básico: O usuário acessa a área de configurações ou perfil no sistema. O sistema disponibiliza a opção "Alterar Senha" ou uma função semelhante. O usuário fornece a senha atual como forma de autenticação. O sistema valida a senha atual fornecida pelo usuário. Se a senha atual é válida, o sistema permite que o usuário insira e confirme a nova senha. O sistema valida a conformidade da nova senha com os critérios de segurança estabelecidos. Se a nova senha é válida, o sistema atualiza a senha associada à conta do usuário. O usuário recebe uma confirmação de que a senha foi alterada com sucesso.

Fluxos Alternativos: Se a senha atual fornecida pelo usuário não é válida, o sistema informa sobre a falha de autenticação e solicita uma nova tentativa. Se a nova senha não atende aos critérios de segurança (por exemplo, comprimento mínimo, caracteres especiais), o sistema informa ao usuário sobre os requisitos necessários.

Pós-condições: A senha associada à conta do usuário é alterada com sucesso.

Regras de Negócio: A nova senha deve atender aos critérios de segurança estabelecidos pela Soy Stocks (S2). O sistema pode impor restrições à frequência de alterações de senha para evitar atividades suspeitas.

Exceções: Se houver problemas técnicos durante o processo de alteração de senha, o sistema deve informar ao usuário sobre a impossibilidade temporária de realizar essa operação.

Notas Adicionais: A alteração regular de senha é uma prática recomendada para manter a segurança da conta, e o sistema pode fornecer lembretes aos usuários para realizar essa ação periodicamente.
